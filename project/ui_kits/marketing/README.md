# Marketing site — UI kit

Recreation of Klyra's public marketing surface, structured around the brand's hero pattern: full-bleed black + halos + display type + pill CTAs with inner-glow.

## Components

- `Nav.jsx` — top nav, white logo, 5 links + right-side pill CTA.
- `Hero.jsx` — full-bleed black hero with two overlapping blue halos, "+20 entreprises" pill, display headline, dual CTA, 4-capability strip.
- `Services.jsx` — 4-up card grid (one highlighted in Bleu K1), numbered `01–04`, eyebrow "→ CE QUE NOUS FAISONS".
- `Proof.jsx` — black proof band with 4 large numerals divided by hairlines.
- `CaseStudy.jsx` — 1.3:1 grid: one big blue case card with mock dashboard + 3 small light case cards.
- `CtaFooter.jsx` — centered display CTA ("Le web n'attend pas") over a radial halo + 4-col footer.

## Open
`index.html` — glues the components into a realistic flow.
