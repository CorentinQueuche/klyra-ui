# Webapp — UI kit

Illustrative webapp chrome showing how the Klyra brand translates into product UI. Klyra builds these kinds of tools for clients (devis, CRM légers, outils métier).

## Components
- `Sidebar.jsx` — 240 px white sidebar with tenant card, nav items, blue CTA card at bottom.
- `TopBar.jsx` — 64 px header with search pill (⌘K), bell icon, primary "+ Nouveau devis" CTA.
- `Dashboard.jsx` — stat cards (4-up), 1.6:1 chart/todos split, recent-deals table with colored status pills.

## Notes
Klyra has no public production UI. Chrome is inferred from the brand book editorial cards (ch. 2.6 "Illustrations + Interfaces") and from the "outil de devis qui remplace 3 Excel" case pattern. **Flag to user if you need a specific real app.**
